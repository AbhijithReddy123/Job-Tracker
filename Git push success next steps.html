What is a Procfile?

A Procfile is a small text file that tells the hosting platform (like Render) how to start your app.
Think of it as giving instructions: “Hey, server, here’s the command to run my web app.”

How to create it

  1. Go to your project folder (the folder where your main .py file lives, e.g., job application tracker).

  2. Create a new file called exactly:

nginx
Procfile

Important: no file extension like .txt — it’s just Procfile.

  1. Open the Procfile and add this line:

makefile
web: gunicorn app:app

What this line means

  • web: → tells Render this is a web service.

  • gunicorn → the server that will run your Python app in production.

  • app:app → points to your Python file and Flask app variable:

Your file nameFlask app variableProcfile line
app.pyappweb: gunicorn app:app
main.pyappweb: gunicorn main:app

Most beginner Flask apps use app = Flask(__name__), so the second app is usually correct.

Once this is done, we can move to updating your Python app to use the dynamic port.

makefile

Option 2: Using Command Prompt / PowerShell

  1. Open your project folder in terminal.

  2. Run this command:

powershell

This creates a Procfile with the correct content.
